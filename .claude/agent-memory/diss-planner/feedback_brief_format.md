Writing briefs must follow the exact format of `chapter2_writing_brief.md`:
- Chapter-level metadata block (target pages, LaTeX file, skeleton version)
- Critical framing note in a blockquote at the top
- Chapter-level intro paragraph spec
- One H2 section per section (§N.N heading), each containing:
  - Target length (words + paragraph count)
  - Purpose statement
  - Paragraph structure (numbered list: topic sentence + key claims + citation placeholders)
  - [TBD] flags block
  - Citations block (IEEE numeric style)
  - Transition out phrase
- Figures/tables summary table
- Cross-reference checklist
- Transitions summary table
- Open questions for Chris block

**Why:** Chris explicitly provided chapter2_writing_brief.md as the format reference and asked for chapter3 to match it exactly.

**How to apply:** Every time a writing brief is produced, verify all of these sections are present. Do not omit the [TBD] flags, transitions summary, or cross-reference checklist.
